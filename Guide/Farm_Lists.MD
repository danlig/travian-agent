# Farm Lists

Source: https://support.travian.com/en/support/solutions/articles/7000064015-farm-lists
Source section: Advanced Gameplay

## What It Teaches

The Farm List is a premium feature included in the Gold Club. It allows you to manage and send multiple raids quickly and efficiently, saving time and improving resource farming.

### Accessing the Farm List

You can access the Farm List through your Rally Point by selecting the Farm List tab.
- If the tab appears gold, you need to purchase the Gold Club to activate this feature.
- Once unlocked, you’ll see all your existing farm lists, organized by village.
- From here, you can edit, send, or create new farm lists.

### Creating a New Farm List
- Go to the Farm List tab in your Rally Point.
- Select Create new list.
- Add a name and assign it to the village you want to send raids from.
- Once created, start adding targets.

You can:
- Add targets manually by entering coordinates.
- Import them from the list of your recent raids sent from the same village.

Each entry in the Farm List defines:
- The target village or oasis.
- The units that will be sent in each raid.

### Sending Raids from a Farm List

After preparing your Farm List, you can:
- Send raids to all targets at once with a single click.
- Open the list manually and start raids for selected targets only.
- Deactivate specific entries to temporarily exclude them from raids (useful if the target now has defenders).

Deactivated entries remain visible and can be reactivated at any time.

### Using Farm Lists for Scouting

You can also use Farm Lists to send scouting missions:
- If you select only spy units in an entry, the system automatically sends a scout mission instead of a normal raid.

### Tip

Farm Lists are ideal for:
- Managing multiple farming routes efficiently.
- Automating raids across large empires.
- Keeping track of which villages are worth raiding again.

The interface allows easy list management — each list shows the number of active targets, last raid times, and quick options to edit, deactivate, or start raids.

## Strategic Implications
- Manage resources, production, and storage capacity as your main constraints: spend before collecting large rewards, and use the NPC merchant only when it unlocks better timing or prevents significant overflow.
- Every military decision must have a clear payoff: profit, defense, conquest, targeted demolition, or diplomatic pressure.
- Plan territory and culture ahead of time: capital, cropper, oases, and expansion slots determine your mid-game trajectory.
