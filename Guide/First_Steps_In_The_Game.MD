# First Steps In The Game

Source: https://support.travian.com/en/support/solutions/articles/7000092522-first-steps-in-the-game

Key early priorities:
- Activate resource bonuses if using gold efficiently.
- Send the hero to the shortest first adventure to obtain early rewards, especially the horse.
- Upgrade resource production steadily and avoid idle construction slots.
- Claim quest rewards deliberately instead of creating warehouse or granary overflow.
- Plan the second village early, preferably a strong cropper or balanced cropper with nearby oasis potential.
- Join a nearby active alliance later instead of creating a weak solo alliance.

For the current account:
- Beginner protection is active, so immediate economy growth is more valuable than troops.
- Keep resources below cap before claiming tasks.
- Continue tutorial task chains that reward resources, culture points, and infrastructure progress.
